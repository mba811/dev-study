# Flask

# 欢迎使用 Flask[](http://www.pythondoc.com/flask/index.html#flask)

![Flask: web development, one drop at a time](http://www.pythondoc.com/flask/_images/logo-full.png)

欢迎阅读 Flask 文档。本文档由很多部分构成。我强烈推荐你们从 [_安装_](http://www.pythondoc.com/flask/installation.html#installation) 开始，接着可以浏览下 [_快速入门_](http://www.pythondoc.com/flask/quickstart.html#quickstart)。比起快速入门（ [_快速入门_](http://www.pythondoc.com/flask/quickstart.html#quickstart) ），[_教程_](http://www.pythondoc.com/flask/tutorial/index.html#tutorial) 内容更详细一些，该文档介绍了如何创建一个完整（尽管很小）的 Flask 应用。如果你想要深入研究 Flask 的话，可以查看 [_API_](http://www.pythondoc.com/flask/api.html#api) 。在 [_Flask 范例_](http://www.pythondoc.com/flask/patterns/index.html#patterns) 章节中描述一些常见的模式。

Flask 依赖着两个外部库： [Jinja2](http://jinja.pocoo.org/2/) 模板引擎以及 [Werkzeug](http://werkzeug.pocoo.org/) WSGI 工具集。本文档并不包含这两个库的文档，查看它们的文档请移步：

  * [Jinja2 文档](http://jinja.pocoo.org/2/documentation/)
  * [Werkzeug 文档](http://werkzeug.pocoo.org/documentation/)
